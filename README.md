# 🌌 Aurora Auth

> A premium Galaxy-themed Login & Signup experience built with **HTML**, **CSS**, and **Vanilla JavaScript**.

Aurora Auth transforms a traditional authentication page into an immersive cosmic experience with stunning animations, futuristic glassmorphism, and delightful micro-interactions. Inspired by modern UI trends from Apple, Framer, and Awwwards, this project showcases advanced frontend design and animation techniques.

## 🚀 Features

- 🌌 Animated galaxy background
- ✨ Flowing aurora lights
- ⭐ Twinkling stars & glowing particles
- ☁️ Floating nebula clouds
- 🪟 Premium glassmorphism UI
- 🌈 Animated neon gradient borders
- 🐰 Cute CSS bunny companion
- 💬 Bunny speech bubbles & hover interactions
- 🔐 Login & Signup forms
- 👁️ Password visibility toggle
- 💪 Password strength indicator
- ✍️ Animated typing headings
- 💎 Floating labels & glowing inputs
- 🎯 Smooth hover & click animations
- ✨ Cursor glow trail
- 🖱️ Mouse parallax effects
- 🌙 Day/Night mode toggle
- 🎉 Success animations with confetti
- ⚙️ Settings panel for customization

- 📱 Fully responsive design
